    │                     ▼
    │               [ Импорт модуля: importlib.import_module ]
    │                     │
    │                     ▼
    │               [ Вызов функции: func(text) ]
    │                     │
    │                     ▼
    │               [ Вывод результата ] ──┐
    │                                       │
    └─ неверный ──> [ Ошибка ] ─────────────┘
                    │
                    ▼
